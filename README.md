# Vehicle Dashboard

Web application built with Streamlit to explore a dataset of vehicle sale ads.

## Features

- Data viewer
- Histogram built with a checkbox
- Scatter plot built with a checkbox
- Vehicle types by manufacturer
- Histogram of condition vs model year
- Price distribution comparison between manufacturers
